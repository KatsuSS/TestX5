`Тестовое задание`

Нужно написать web-сервис - кабинет рекламодателя.
Желательно использовать только стандартную библиотеку Python, Django и DRF.

`Билд:` docker-compose build

`Запуск:` docker-compose up -d

`Создать супер юзера:` docker-compose exec -it web uv run python manage.py createsuperuser
